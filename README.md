# Calculator-Program

This repo has 3 Py files demostrating below,

1. Basic Calculator - Creates a calculator that takes two numbers and an operator (+, -, *, /) and shows the result.
2. Add Result Check - Extended 1: After showing result, checking if it's positive, negative, or zero and print accordingly.
3. Student Grade Calculator - Calculates student grade based on marks in 3 subjects.
